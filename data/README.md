## Dataset Information

This project uses the **CrossCheck smartphone-based digital phenotyping dataset**, which is publicly available on Kaggle.

Dataset link:
https://www.kaggle.com/datasets/dartweichen/crosscheck

To run this project:
1. Download the dataset from Kaggle.
2. Extract the files into this `data/` directory.
3. Ensure the file paths match those referenced in `analysis.ipynb`.
